K-means clustering on images

Step 1: Compile

Go to the path where the java files (Main.java and KMeans.java) are present.
>javac *.java

Step 2: Run (Please give the full path)

>java Main <Input image path> <Output folder path> <K-value>

Sample Input: 

>java Main D:\\Sem2\\ML\\HW3\\Penguins.jpg D:\\Sem2\\ML\\HW3\\Penguins_output 2

Sample Output:

If k-value is large, the program takes some time(approx 5 mins) to complete. Please be patient.

For the ouput image, please see the folder D:\\Sem2\\ML\\HW3\\Penguins_output

Size of the input(original) Penguins.jpg is 759.6044921875 KB
Size of the output(compressed) Penguins image for k=2 is 83.65234375 KB

Image Compression Ratio = Original Image Size / Compressed Image Size = 9.08049264534205
